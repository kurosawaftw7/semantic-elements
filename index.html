Semantic Elements in HTML

A Brief Lesson on The Subject

By

Jeff H

Definition

A semantic element is a piece of HTML code which communicates to both the website and developer the intention and purpose of that piece of code. Semantic elements act as both a programming language and reference for other programmers.

Reasons to Use Elements:

Reasons for using semantic elements include:

  • Accessability - help assistive technology interpret a webpage
  • Searchability - help computers search for your content
  • Internationalization - help people using other languages translate your website
  • Interoperability - help other programmers understand a webpage
